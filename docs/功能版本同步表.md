## 性能优化

| 效果 | Web 方式 | Android 优化 |
|------|---------|-------------|
| 涟漪 | CSS ripple 触发重绘 | RippleDrawable 原生 GPU |
| 玻璃 | backdrop-filter 每元素 | 单层渐变 drawable |
| 辉光 | CSS box-shadow 触发重绘 | elevation + 半透明底色 |
| 缓动 | CSS cubic-bezier | PathInterpolator 原生线程 |
| 背景 | CSS radial-gradient | 单层 shape drawable |
| 触感 | Navigator.vibrate() | VibrationEffect 精确振幅 |
