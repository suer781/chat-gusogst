## 变更清单

### 1. MaterialAnimator v2
- 修复 `applyButtonPress` 返回 `false` 让 onClick 仍能触发
- 新增 Material You 缓动曲线：`EASE_DECELERATE`、`EASE_ACCELERATE`（PathInterpolator）
- 新增 `animateIndicator` — 导航指示器 spring bounce 动画（替代 200ms 减速）
- 新增 `applyThemeTransition` — 主题切换 0.6s 交叉渐变
- 新增 `setAmbientBackground` — 环境光背景（单层 GPU drawable）
- 使用原生 `RippleDrawable`（零 Java 开销），比 Web CSS ripple 更快

### 2. HapticsHelper v2
- 从 3 种模式扩展到 15 种语义化触感：`light`、`medium`、`heavy`、`success`、`error`、`warning`、`selectionStart`、`selectionChanged`、`selectionEnd`、`glassTap`、`glassPress`、`glassSlide`、`sendPulse`、`unfold`、`sliderTick`
- 精确振幅控制（替代 Web 固定 pattern 数组）
- 节流机制：glassSlide 60ms，selectionChanged 80ms

### 3. HdrHelper v2
- 新增 `indicatorGlow`、`inputFocusGlow`、`buttonGlow` 色值
- `applyIndicatorGlow` — 导航指示器 HDR 辉光
- `applyInputGlow` — 输入框 focus 辉光
- 使用 elevation 阴影替代 CSS box-shadow（GPU 友好）

### 4. MainActivity
- 环境光背景：`MaterialAnimator.setAmbientBackground` + `R.drawable.bg_ambient`
- Nav 点击触感：`haptics.glassTap()` 替代 `HapticFeedbackConstants.VIRTUAL_KEY`
- 导航指示器 spring：`MaterialAnimator.animateIndicator()` 替代 `view.animate().x()`
- 玻璃效果 v2：三层渐变（白→蓝紫→深色）匹配 Web `[data-glass="on"]`
- 主题切换：`MaterialAnimator.applyThemeTransition()` 6s 交叉淡入淡出

### 5. MessageAdapter
- HDR 气泡辉光：user 和 assistant 气泡都调用 `HdrHelper.applyBubbleGlow`
- 按钮动画：copy/regen/delete 按钮都有 `MaterialAnimator.applyButtonEffects`（涟漪 + scale）

### 6. ChatFragment
- 传递 `hdrEnabled` 和 `isDark` 到 adapter
- 发送按钮：`MaterialAnimator.applyButtonEffects`
- 页面进入动画：`MaterialAnimator.viewEnter`

## 性能优化对比

| 效果 | Web 方式 | Android 方式 | 优势 |
|------|---------|-------------|------|
| 涟漪 | CSS ripple 触发布局 | `RippleDrawable` 原生 | 零 Java 开销，GPU 合成 |
| 玻璃 | `backdrop-filter` 每元素 | 单层渐变 drawable | 无 per-element 重绘 |
| 辉光 | CSS `box-shadow` | elevation + 半透明底色 | 不触发重绘 |
| 缓动 | CSS `cubic-bezier` | `PathInterpolator` | 原生动画线程 |
| 背景 | CSS `radial-gradient` | 单层 shape drawable | GPU 单次合成 |
| 触感 | `Navigator.vibrate()` | `VibrationEffect` | 振幅精确控制，省电 |

## 尚未实现

- RenderEffect blur（API 31+）— 需等 minSdk 提升到 31 才能用
- 噪声纹理 — 开销高且视觉增益有限，暂不移植
- 过度滚动辉光颜色 — EdgeEffect 反射方案不稳定，待 API 31+
- Fragment 间切换动画（viewEnter/viewExit）— 需在 `FragmentTransaction.setCustomAnimations` 中集成
